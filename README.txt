функция downloadFile пытается загрузить файл по url, если загрузка проходит
успешно то файл сохраняется локально, если неудачно повторяется попытка 
до истечения времени ожидания 

функция scrapeLinks собирает все ссылки на файлы которые имеют расширение 
с помошью библеотеки colly

функции logFailure и logSuccess открывают файлы логов и записывают туда имена файлов
